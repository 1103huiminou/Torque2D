virtual int GuiTextListCtrl::addRow ( ID  ,
text  [, row] 
)
virtual

Use the addRow method to add a new entry to the text list.

Parameters
IDThe integer ID to assign to this entry. May be between 0 and inf and can be repeated. i.e., multiple entries may have the same ID.
textThe text to display for this entry in the list.
rowAn optional integer value representing the position at which to add this entry, where 0 is the top of the list, 1 is after them first entry (if any), etc.
Returns
Returns the row number of the new entry.
