typedef type string

string is a series of characters of arbitrary length.

You represent a string by surrounding it with double quotes, such as "some string". For example, "this is a string" but so is "this" or even "".

typedef type Vector

Vector is a string of any number of values separated by spaces between each.

For example, "1 2 3 4" is a 4-vector, as is "a b c d". You can create a vector from scalars (single numbers) by using the SPC operator.

"50" SPC "30.5"

You can also combine smaller Vectors with the SPC operator:

"a b" SPC "c d"
typedef type Vector2

Vector2 is a string of two numbers separated by a space.

For example, "50 30.5" is a 2-vector with an x value of 50 and a y value of 30.5. You can create a vector from scalars (single numbers) by using the SPC operator.

50 SPC 30.5
