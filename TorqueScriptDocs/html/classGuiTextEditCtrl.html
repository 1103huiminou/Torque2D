void GuiTextEditCtrl::setCursorPos ( newPos  )

Use the setCursorPos method to set the current position of text cursor to newPos. If the requested position is beyond the end of text, the cursor will be placed after the last letter. If the value is less than zero, the cursor will be placed at the front of the entry.

Parameters
newPosThe new position to place the cursor at, where 0 is at the beginning of the line, 1 is after the first letter, and so on.
Returns
No return value.
